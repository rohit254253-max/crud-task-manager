# 🤖 AI Chatbot using Flask

This is a simple AI Chatbot project built using Python Flask, HTML, CSS, and API integration.

## 🚀 Features

- User-friendly chatbot UI
- Flask backend
- API integration
- Dynamic bot responses
- Responsive design

## 🛠️ Tech Stack

- Python
- Flask
- HTML
- CSS
- Requests Library
